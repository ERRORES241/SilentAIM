# SilentAim for The Long Dark

[![MelonLoader](https://img.shields.io/badge/MelonLoader-0.7.2-blue.svg)](https://github.com/LavaGang/MelonLoader)
[![ModSettings](https://img.shields.io/badge/Dependency-ModSettings-orange.svg)](https://github.com/zeobviouslyfakeacc/ModSettings)
[![Version](https://img.shields.io/badge/Version-1.0-green.svg)](#)

A simple yet powerful silent aim mod for **The Long Dark**. Includes visibility checks, filters, custom FOV, and support for multiple body parts.

---

### Features
*   **Visibility Checks** — Targets only those you can actually see.
*   **Filters** — Ignore specific animals or targets.
*   **Custom FOV** — Fully adjustable field of view circle.
*   **Target Selection** — Aim at 4 specific bones: `Head`, `Neck`, `Torso`, `Hips/Limb`.
*   **Visuals** — Target info, tracer lines, and FOV circle.

### Supported Weapons
Now supports projectiles!
*   **Rifle**
*   **Revolver**
*   **Bow** (Arrows)
*   **Stones** (New)
*   **Flare Gun** (New)

---

### Default Hotkeys
| Action | Key |
| :--- | :--- |
| **Master Switch** | `F2` (Configurable) |
| **Choose Bone** | `F3` (Configurable) |
| **FOV +/-** | `CTRL` + `Scroll Up/Down` |

### Requirements
1.  **MelonLoader** (v0.7.2+)
2.  **ModSettings**
3.  **The Long Dark** (v2.54+)

### Installation
1.  Ensure you have the requirements installed.
2.  Place `SilentAIM.dll` into your `Mods` folder.
3.  Go annihilate...
